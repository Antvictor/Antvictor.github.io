java 进阶 -jvm

生成内存溢出快照：在runConfigurations中的VM arguments添加：-XX:+HeapDumpOnOutOfMemoryError 。

-Xss规定了每个线程堆栈的大小。一般情况下256K是足够了。影响了此进程中并发线程数大小。

-Xms初始的Heap的大小。

-Xmx最大Heap的大小。

在很多情况下，-Xms和-Xmx设置成一样的。这么设置，是因为当Heap不够用时，会发生内存抖动，影响程序运行稳定性。
