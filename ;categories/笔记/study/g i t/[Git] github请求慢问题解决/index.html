最简单的办法，做映射

​ 先查询github的ip地址，然后在本地做映射，可以有效提升github访问速度

​ ip查询 : http://tool.chinaz.com/

​

如上图所示，13.229.188.59 搭载在新加坡，并且ttl只有8，所以我们就映射这个ip。

找到hosts文件，如mac 在/etc/ 文件夹下，

然后本地请求github.com 时 其实是请求了ip地址，而这个ip地址的延迟是很低的，因此可以有效的提高访问速度。
