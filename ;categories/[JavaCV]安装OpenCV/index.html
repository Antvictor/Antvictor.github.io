安装cmake

下载cmake

  • 首先进入下载界面下载cmake，然后安装
  • 根据 tools-> how to install for Command line Use。 将cmake设置为命令行可用
    • 根据第一种方式，将 /Application/cmake.app/bin 配置到 /root/.bash_profile 中即可
  • cmake --version 测试是否成功
